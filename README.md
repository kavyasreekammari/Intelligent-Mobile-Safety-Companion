# Intelligent Mobile Safety Companion for Mine Workers

This project detects helmet and PPE using YOLOv8 and Flask.

## Features
- Helmet detection
- Safety checklist
- Incident reporting
- Web dashboard

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- YOLOv8

## How to Run
1. Install requirements
2. Run: python app.py
